This is a whatsapp automation tool designed in Python. It can be used to send important message at a specific time so that we don't have to disturb our busy life just because of sending a message at a specific time.    It is easy to use only we have to write the mobile number to which we have to send the message and the special time for the message. In this i have used Python library which is used to send message through whatsapp. In short only we have to do is open our whatsapp through whatsapp web in the default browser. The rest will be handled by the program itself. Enjoy
